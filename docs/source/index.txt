==============
Personal Stack
==============

The goal of the  "Stack" project is to collect a number of "quality
working defaults," for programs like `Emacs`_ and `StumpWM`_ to
provides users with a starting point for their own configuration and
customization development. While both of these programs (and others)
are very powerful tools, without some amount of configuration these
tools are not particularly useful.

I'm producing these configurations from the configurations I use every
day. The methodology that I used for extracting and packaging this
configuration is available in `this repository`_, and

Continue reading for more information about the components of the
stack.

-- `tychoish`_

.. _`StumpWM`: http://stumpwm.org/
